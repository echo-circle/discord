Echo Circle

About the Server

Echo Circle is a community server where you can chill and game. There is always someone to talk to, and it is a very laid back server. There are a lot of things to entertain yourself here. One of the biggest things is definitely our main, or "general", chat. It is very active, and there is almost always someone there using it. This is vital in good servers. If there is not anyone using the server, then it is hard to enjoy by yourself, no matter how good the design and thought put into the server is. Another very nice thing about our server is our wide array of bots. We have some bots for moderating (which keeps our server safe and gets rid of rule-breakers), we have bots for entertainment, and bots for utility that can track server statistics, give out roles, etc.

Staff Layout

The server has a very strong staff team that does a good job at keeping the server safe and fixes problems. We have our Head Administrators, Administrators, Head Moderators, Moderators, and Trial Moderators. We have high expectations for our staff, and don't want people who wish bad on the server to acquire special permissions to do bad things. To prevent this, those who are accepted start as "Trial Moderators". We watch them to see how well they do before promoting them and giving them permissions of a full moderator. The moderators watch over the server. They are the ones that ban/kick/warn those who break the rules. They are required to be active so that we know they are watching the chats. The head moderators do the same thing, but they also watch over the moderators.
